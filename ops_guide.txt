Operaciones aceptadas por la terminal
--add [comando] [pathToFile]
-si no existe
Agrega el comando a la lista de comandos con el path a ejecutar cuando este comando sea llamado
-si existe
actualiza el path del comando
--rm [comando]
Elimina el comando si existe de la lista de comandos, si no lanza error
--[comando]
Ejecuta el comando si existe, si no lanza error
--ls 
Lista los comandos disponibles, si no existiesen muestra texto de que no existen comandos disponibles
--exit 
Sale de la ventana de la terminal
